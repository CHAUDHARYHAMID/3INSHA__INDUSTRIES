# **App Name**: Insha Industries Website

## Core Features:

- Content Display: Display content from inshaindustries.com.
- Image Gallery: Showcase images related to Insha Industries.
- Contact Form: Provide a contact form for users to send inquiries.

## Style Guidelines:

- Primary color: A deep, authoritative blue (#1A434D), evoking stability and trust, mirroring the industrial focus. The color choice uses blue's associations with competence and quality, core brand attributes for the company.
- Background color: A light, desaturated blue (#EBF2F4), providing a clean and professional backdrop.
- Accent color: A muted green (#6A8A82), employed to highlight key interactive elements and CTAs, providing contrast against the primary color.
- Font: 'Inter', a sans-serif font, will be used for both headings and body text. It conveys modernity and clarity.
- Clean and organized layout to ensure a professional and trustworthy appearance.  Maintain clear visual hierarchy and logical flow to present information.
- Use industry-related icons, in the accent color.
